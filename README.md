# 🌐 [David Sepkitt | Personal Portfolio](https://davidsepkitt.netlify.app/)

A modern, responsive personal portfolio website showcasing my journey as an **ICT Application Development student** and aspiring **Full Stack Software Developer**. This portfolio highlights my technical skills, featured projects, education, and experience while providing an easy way for recruiters, employers, and fellow developers to learn more about me and get in touch.

Designed with a premium user experience in mind, the website features smooth animations, responsive layouts, interactive UI elements, and a clean, modern aesthetic inspired by today's leading technology companies.

---

## ✨ Features

* 🎨 Modern and responsive design
* 🌙 Premium dark theme with gold accent colours
* ✨ Interactive cursor spotlight effect
* 🌌 Animated ambient background particles
* 📱 Fully responsive across desktop, tablet, and mobile devices
* 🚀 Smooth scrolling and polished animations
* 🎯 Active navigation highlighting (Scroll Spy)
* 💼 Dedicated sections for:

  * About Me
  * Skills
  * Featured Projects
  * Experience & Education
  * Contact
* 📬 Functional contact form powered by Resend
* ⚡ Fast performance and optimized loading experience

## 🚀 Running Locally

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
```

Navigate into the project:

```bash
cd YOUR_REPOSITORY
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
